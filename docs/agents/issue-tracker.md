# Issue tracker: GitHub

Issues and PRDs for this repository live in GitHub Issues for `jeremyzejunchen/finance-tracker`.

## Conventions

- Create, read, update, label, and close issues through the configured GitHub connector when available.
- Apply the `ready-for-agent` label only to fully specified, agent-ready work.
- Pull requests are not a triage surface for this repository.

## When a skill says "publish to the issue tracker"

Create a GitHub issue in `jeremyzejunchen/finance-tracker`.

## When a skill says "fetch the relevant ticket"

Read the GitHub issue and its comments before implementation.
